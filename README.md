## 技術整合

設計：
- Figma
- Bing Image Creator

前端：
- Tailwind
- jQuery
- tilt.js
- SVG Animation

<br>

## 功能
本次開發為純前端，無串接任何後端服務，  
因此表單行為不會發出任何網路請求。  

- 導覽列：點擊連結快速定位到區塊
- 大圖標語：SVG 變形動畫與快速定位到小額贊助區
- 政見主張：jQuery 實作輪播圖
- 活動快報：CSS 動畫，以 checkbox 狀態改變樣式
- 小額贊助：彈出視窗
- 服務信箱：彈出視窗
- Top 按鈕：返回螢幕上方

如有功能異常歡迎聯絡，感激不盡！
